- 주어진 문제를 여러 개의 하위 문제로 나누어 푸는 방법
- 같은 하위 문제가 등장하면 계산하지 않는다.
- 2가지 DP 방법

    ***1) Top Down Approach : `Memoization` 사용***

    ***2) Bottom Up Approach : `Tabulation`(encode an order in which solutions are calculated) 사용***
